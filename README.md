# 📊 Employee Performance Predictor (AI-Powered HR Analytics System)

## 🚀 Project Overview
This project is an end-to-end Machine Learning system that predicts employee performance levels (High / Medium / Low) using HR analytics and behavioral data.

It is designed as an **industry-level HR decision support system** for companies to improve workforce productivity, reduce bias, and optimize talent management.

---

## 🎯 Problem Statement
Companies struggle to:
- Identify high-performing employees early
- Detect underperforming employees
- Make unbiased promotion decisions
- Optimize training and workforce planning

This system solves these challenges using Machine Learning.

---

## 🧠 Key Features
- 📊 Employee Performance Prediction (H/M/L)
- 📈 Feature Importance Analysis
- ⚖️ Fairness & Bias Detection
- 💼 HR Recommendation Engine
- 📉 Productivity & Business Impact Estimation
- 📊 Streamlit Interactive Dashboard
